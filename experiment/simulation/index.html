🖥️ 5G SBA Simulation — Help

🚀 Simulation Steps
  1. Open Docker Terminal → run docker compose -f docker-compose.yml up -d to deploy the core network (AMF, SMF, UPF, NRF, etc.).
  2. Deploy the gNB → docker compose -f docker-compose-gnb.yml up -d
  3. Deploy UEs → docker compose -f docker-compose-ue.yml up -d
  4. Monitor all services → watch docker compose -f docker-compose.yml ps -a
  5. Click 📡 PDU Session button to step through the 5G UE registration procedure.
  6. Click ✓ Validate to verify your network topology is correctly configured.
  7. Tear down core network → docker compose -f docker-compose.yml down
